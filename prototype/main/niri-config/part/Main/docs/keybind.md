## System

| Keybind                | Action                     | Command                             |
| ---------------------- | -------------------------- | ----------------------------------- |
| `Alt + Shift + x`      | Exit                       | `wlogout`                           |
| `Alt + Ctrl + x`       | Logout                     | `quit`                              |
| `Alt + Ctrl + Delete`  | Logout                     | `quit`                              |

> run `yay -Sy --needed wlogout` to install `wlogout`


| Keybind                | Action                     | Command                             |

| `Alt + Ctrl + z`       | Monitors Power Off         | `power-off-monitors`                |
